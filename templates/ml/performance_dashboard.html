Methodology & Interpretation
How It Works
  1. Daily Snapshots: Every day at 1:00 AM, the system generates predictions for all open work orders and saves them to S3.
  2. Realized Performance: Once work orders complete, the system compares predicted vs actual completion times.
  3. Error Metrics:
    • MAE (Mean Absolute Error): Average prediction error in days. Lower is better.
    • RMSE (Root Mean Squared Error): Penalizes larger errors more heavily. Useful for detecting outliers.
  4. Confidence Intervals: 95% CI using t-distribution accounts for sample size variability (n shown above each point).
  5. Trend Analysis: Linear regression detects if model is improving, degrading, or stable over time.
What to Watch For
Model Drift: If MAE increases significantly or trend is degrading, the model may need retraining.
Convergence: Stable or improving MAE indicates the model is well-calibrated.
Sample Size: Larger n (shown above each point) means more reliable error estimates.
